# Neo TCG (prototype)

Browser-based TCG vertical-slice (player vs simple AI).

Features:
- 3 mana colors (R/G/B)
- auto mana per turn
- summoning sickness
- units with attack/health/cost

## Run
Open `index.html` in a browser.
